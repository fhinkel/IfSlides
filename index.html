Exercise: NodeJS Hello World!

  • Install node http://nodejs.org/, to test your installation, type in a terminal: 
    node --version
  • Use any editor to write code
  • Save the file as server.js
  • In a terminal or shell, in the same directory as server.js, type 
    node server.js
  • Check by opening http://localhost:8080 in your browser

Chat Server

  • Install IntelliJ Community Edition http://www.jetbrains.com/idea/download/
  • Clone the chat server repo https://github.com/fhinkel/SimpleChatServer
  • Start the server 
    node js/server.js
    and check that it is running in your browser on http://localhost:1337

Exercise: Make tests green!

  • Install all necessary node modules by running 
    npm install
  • Run the tests: 
    npm test
  • Fix the tests so that both are green!
  • Push your changes to the remote repository
  • Fork the repository https://github.com/fhinkel/SimpleChatServer on Github.
  • Give your pairing partner permission to push changes to it
  • Clone your forked repository to a new directory (or delete the SimpleChatServer directory)
  • work on your forked repo, we will merge changes when the first feature is complete

Let's program! And remember, test first!

  • Use the express framework, http://expressjs.com/
  • The server should respond with Hello {name}!, when you open http://localhost:8080/name
  • Create an input field instead of using the url for the correct greeting
  • Instead of a greeting, display all messages sent to the server, a new message should be shown to all clients
  • Users should enter their nickname and messages should be displayed with their name
  • Extra features: Smiley support, private rooms,
